# @airbnb-extended/eslint-config

Shared ESLint configuration for the Airbnb Extended monorepo.

## Usage

### Eslint Configuration

```js
// eslint.config.js
import config from '@airbnb-extended/eslint-config/base';

export default config;
```

### Internal Configuration

```js
// eslint.config.js
import config from '@airbnb-extended/eslint-config/internal';

export default config;
```

### Root Configuration

```js
// eslint.config.js
import config from '@airbnb-extended/eslint-config/root';

export default config;
```
