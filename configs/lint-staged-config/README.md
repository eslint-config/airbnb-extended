# @airbnb-extended/lint-staged-config

Shared Lint Staged configuration for the Airbnb Extended monorepo.

## Usage

### Base Configuration

```js
// lint-staged.config.js
import config from '@airbnb-extended/lint-staged-config/base';

export default config;
```

### Root Configuration

```js
// lint-staged.config.js
import config from '@airbnb-extended/lint-staged-config/root';

export default config;
```
